# Changelog

## [0.0.18]

### Added
- Node.js SQLite wrapper using native `node:sqlite` module
- Core database operations: open, close, execute, query, queryOne
- Transaction support with automatic commit/rollback
- Prepared statements support
- In-memory and file-based database support
- Comprehensive test suite with 100+ test cases
- TypeScript type definitions
- Full API documentation in README
- Example code demonstrating all features

### Changed
- Updated package.json with correct project metadata
- Renamed main export from template-project to node-sqlite

### Requirements
- Node.js >= 22.14.0 (for native sqlite module support)
